Steps to download and install node in ubuntu

  1. Step 1: Download latest or recommended node .tar.xz file from https://nodejs.org/en/ or you can download node version 14.15.5 (.tar.xz file) directly from here -> https://nodejs.org/dist/v14.15.5/node-v14.15.5-linux-x64.tar.xz
  2. Step 2: Go to the directory in which (.tar.xz file) is downloaded. In my case --> /Download directory
  3. Step 3: Update System Repositories sudo apt update
  4. Step 4: Install the package xz-utils sudo apt install xz-utils
  5. Step 5: To Extract the .tar.xz file sudo tar -xvf name_of_file In my case --> sudo tar -xvf node-v14.15.5-linux-x64.tar.xz
  6. Step 6: sudo cp -r directory_name/{bin,include,lib,share} /usr/ In my case --> sudo cp -r node-v14.15.5-linux-x64/{bin,include,lib,share} /usr/
  7. Step 7: Check the node version node --version Result In my case -> v14.15.5
